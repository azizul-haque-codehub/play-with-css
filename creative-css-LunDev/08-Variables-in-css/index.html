Variable in css

In CSS, variable mean not other than like Programming language ,
In css , variable also contain css property value like
syntax : --variable-name : value or
@property --variable-name{ prop:value }
--primary-text: #f0f

Disadvantage of --variable in css

  1. Inheritance cannot be prevented
    for example:
    :root { --box-bg: #003; }
    body { --box-bg: #0f0; }
    main{ --box-bg: red; }
    .box { width: 200px; height: 200px; border: 2px solid;
    background-color: var(--box-bg);
    border-radius: 5px; }
    here box bg will be red not from above
  2. Variable value can be reassign and set any others value
    for example:
    :root { --box-bg: #003; }
    body { --box-bg: #0f0; }
    div{ --box-bg: 0.5; }
    .box { width: 200px; height: 200px; border: 2px solid;
    background-color: var(--box-bg);
    border-radius: 5px; }
    here box bg will be 0.5 which are not formal value of bg
